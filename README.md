# 🎨 UX/UI Projects

[![Figma](https://img.shields.io/badge/Figma-Designs-%23F24E1E.svg?style=for-the-badge&logo=figma&logoColor=white)](https://figma.com)

**Мои работы по UX/UI-дизайну** — от мобильных приложений и лендингов до интернет-магазинов, игр и UI-китов.

---

## ✨ О репозитории

- **Типы проектов**: мобильные приложения, landing pages, e-commerce, корпоративные сайты, тесты, игры, упаковка и UI-киты.
- **Подход**: чистый минимализм, внимание к деталям, продуманные user flow и микровзаимодействия.
- **Формат**: все проекты доступны по прямым ссылкам на Figma (с возможностью просмотра прототипов).
